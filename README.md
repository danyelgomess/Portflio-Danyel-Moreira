# Portfolio-Danyel-Moreira

Portfólio profissional com projetos práticos em Cloud Computing, DevOps e Desenvolvimento Full Stack. Este repositório reúne laboratórios, arquiteturas em AWS, automação de processos, versionamento com Git/GitHub, deploy de aplicações web e desenvolvimento de sistemas completos, abrangendo front-end, back-end e infraestrutura em nuvem.

Os projetos demonstram conhecimentos em serviços AWS, integração e entrega contínua (CI/CD), monitoramento, boas práticas de segurança, infraestrutura como código e princípios do AWS Well-Architected Framework, além da construção e publicação de aplicações web funcionais.
